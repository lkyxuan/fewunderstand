## 这是一个测试文件

用来测试 Obsidian Git 插件的自动同步功能：

1. 文件创建后，应该会自动提交和推送
2. 编辑后停止一会，应该会自动同步
3. 如果看到这个文件出现在 GitHub 上，说明自动同步成功了！

现在您可以：
1. 等待看是否自动同步
2. 编辑这个文件测试实时同步
3. 检查 GitHub 仓库是否更新
